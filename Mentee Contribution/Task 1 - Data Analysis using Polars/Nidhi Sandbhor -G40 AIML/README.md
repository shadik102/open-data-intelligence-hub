# Travel Industry Analytics Platform

An interactive data analytics dashboard built using Python and Streamlit to analyze tourism and accommodation trends across different countries and years.

## Features
- Tourism trend visualization
- Country-wise analysis
- Interactive charts and dashboards
- Data cleaning and preprocessing
- Future trend forecasting

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn

## Files
- app.py
- analysis.py
- clean.py
- requirements.txt

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
